# 🏋️ Gym Project

[![CI Gym Project](https://github.com/DevJhoow/Gym-Project/actions/workflows/ci.yml/badge.svg)](https://github.com/DevJhoow/Gym-Project/actions/workflows/ci.yml)

Um sistema web criado para ajudar pessoas que treinam a **acompanhar sua evolução física de forma simples e rápida**.

A ideia do projeto é aplicar na prática os conhecimentos de **JavaScript**, organizando tudo em um único projeto que evolui conforme novos conceitos são aprendidos.

---

# 🎯 Problema que o projeto resolve

Muitas pessoas que treinam acabam se perdendo em informações como:

- peso atual
- evolução corporal
- cálculo de IMC
- metas de treino
- organização de dados

Normalmente essas informações ficam espalhadas em:

- papel
- anotações no celular
- aplicativos complexos

O **Gym Project** busca resolver isso com uma ideia simples:

> O usuário só precisa se preocupar em **treinar**.  
> O sistema cuida de **organizar, calcular e mostrar a evolução**.

---

# 💡 Proposta do sistema

O objetivo do projeto é criar um sistema que:

- facilite o **acompanhamento da evolução física**
- centralize informações importantes do usuário
- automatize tarefas que normalmente são feitas manualmente
- mostre dados de forma **rápida e clara**

No futuro o sistema poderá utilizar:

- **IA para sugestões de treino e evolução**
- **APIs externas** para dados e cálculos
- **testes automatizados** para garantir estabilidade do sistema

---

# ⚙️ Funcionalidades atuais

✔ Cadastro de usuário  
✔ Login e logout  
✔ Controle de sessão  
✔ Perfil individual por usuário  
✔ Registro de peso e objetivos  
✔ Cálculo de IMC  
✔ Interface simples e rápida  

Cada usuário possui **seus próprios dados**, garantindo que as informações não se misturem entre contas.

---

# 🧠 O que estou praticando nesse projeto

Este projeto funciona como um laboratório de aprendizado para praticar:

- JavaScript na prática
- manipulação de DOM
- organização de código
- lógica de sistema
- armazenamento de dados com `localStorage`
- separação de responsabilidades no código
